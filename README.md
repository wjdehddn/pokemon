## 🔧 Installation & Setup  
```bash
# 1. 저장소 클론
git clone https://github.com/wjdehddn/pokemon.git

# 2. 프로젝트 디렉토리 이동
cd blog

# 3. 패키지 설치
npm install

# 4. 프로젝트 실행
npm start
